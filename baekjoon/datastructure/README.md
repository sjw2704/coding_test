# Datastructure

## 핵심 패턴
- 자료구조 선택 전에 입력 형식과 제약을 정확히 분석.
- 시뮬레이션 시 **저장할 상태(what)**와 **변화 규칙(when/how)**을 명확히 정의.

## 자주 터지는 실수
- >>와 getline 혼용 시 개행 문자 처리 누락.
- 큐·스택·덱 회전, 그래프 좌표 이동 등에서 인덱스 모듈러 연산 실수.

- 11098_첼시를도와줘 — 문자열 파싱
- 5635_생일 — 정렬 기준 커스텀
- 28215_대피소 — 브루트포스+구현 (tags: brute-force)
